Judul itu Penting

Mesin pencari menggunakan judul untuk mengindeks struktur dan konten halaman web Anda.

Pengguna sering membaca halaman dengan judulnya. Penting untuk menggunakan judul untuk menunjukkan struktur dokumen.

Judul <h1> harus digunakan untuk judul utama, diikuti oleh judul <h2>, lalu yang kurang penting <h3>, dan seterusnya.

Catatan: Gunakan judul HTML untuk judul saja. Jangan gunakan judul untuk membuat teks BESAR atau tebal.

Bagaimana cara melihat sumber HTML?

Pernahkah Anda melihat halaman Web dan bertanya-tanya, "Hei! Bagaimana mereka melakukan itu?"

Melihat sumber kode HTML:

Klik kanan di halaman HTML dan pilih "Lihat Sumber Halaman" (di Chrome) atau "Lihat Sumber" (di Tepi), atau yang serupa di browser lain. Ini akan membuka jendela yang berisi kode sumber HTML halaman.

Periksa Elemen HTML:

Klik kanan pada elemen (atau area kosong), dan pilih "Inspect" atau "Inspect Element" untuk melihat elemen apa yang dibuat (Anda akan melihat HTML dan CSS).Anda juga dapat mengedit HTML atau CSS saat itu juga di panel Elemen atau Gaya yang terbuka.
