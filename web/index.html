1. Introduction

Apa itu HTML?

HTML adalah standard markup languanges untuk membuat halaman web.

  • HTML adalah singkatan dari Hypertext Markup Languange.
  • HTML mendeskripsikan struktur dari halaman web.
  • HTML terdiri dari serangkaian elemen.
  • Elemen <html> memberitahu browser bagaimana content harus ditampilkan.
  • Elemen <html> diwakili oleh tag
  • HTML memberi label - label pada konten seperti heading, Paragraf, table, dan lain sebagainya
  • Browser tidak menampilkan tag HTML, tetapi menggunakannya untuk membuat konten pada halaman.

HTML Tags

HTML tags adalah nama elemen yang dikelilingi oleh tanda sudut :

  • Tag <html> biasanya berpasangan.
  • Tag pertama dalam pasangan adalah tag awal, tag kedua adalah tag akhir.
  • Tag akhir ditulis seperti tag awal, tetapi dengan garis miring dimasukkan sebelum nama tag (<html> </html>)
